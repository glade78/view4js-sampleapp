
routes will be set in ViewManager
eventRoutes will be set in Individual Navigator
EventRoutes are for forward navigation only . 
EventRoutes does not provide order of view loading but only target view for that navigation event.
Combination of Path+NavEvent also unique and must not be duplicate.

Logic For TabNavigator :

1) Each Tab is ViewStack 
2) Each Tab have its own history
3) On Click of Tab Menu 
    -> Disable Current Tab and ViewStack, 
    -> Disable Current View in ViewStack
    -> Get ID of New Tab / ViewStack 
    -> Enable ViewStack /Tab
    -> Enable Current View / Create default view 

4) This is not Navigation Event
